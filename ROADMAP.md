# Roadmap — BoSP Distortion

Short-term (now)
- Finish WaveShaper modes and test coverage (Soft, Tube, Hard, Fold).
- Add Dry/Wet and Tone filter.
- Improve editor layout and avoid overlap across common window sizes.

Medium-term
- Add oversampling with configurable rate and anti-aliasing.
- Add presets and parameter snapshot management.
- Add automated audio regression tests where feasible.

Long-term (v1.0)
- Polished UI and complete documentation.
- Release packaging for VST3 installers.
- Performance tuning and platform QA across hosts.
